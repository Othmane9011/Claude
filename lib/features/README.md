# Features

Ce dossier contient les différents modules de l'application VetHome.

## Modules disponibles

- **admin** - Administration
- **adopt** - Adoption d'animaux
- **auth** - Authentification
- **boarding** - Pension pour animaux
- **bookings** - Réservations
- **gate** - Portail d'accès
- **grooming** - Toilettage
- **home** - Page d'accueil
- **map** - Carte des services
- **pets** - Gestion des animaux
- **petshop** - Boutique pour animaux
- **pro** - Espace professionnel
- **profile** - Profil utilisateur
- **providers** - Fournisseurs de services
- **reviews** - Avis et évaluations
- **vets** - Vétérinaires
